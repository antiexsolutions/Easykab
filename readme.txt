A fully customizable taxi booking plugin for WordPress integrated with Google Maps, WooCommerce, and Elementor.

== Description ==

Smart Taxi Booking provides a complete ride booking solution with features like:

- Google Maps integration for pickup/drop/waypoints
- Distance-based dynamic fare calculation
- Multi-vehicle pricing slabs
- Driver auto-assignment based on pickup radius
- Meet & Greet and Waiting time pricing
- WooCommerce checkout integration
- Elementor compatibility
- Email notifications for customer & driver
- Mobile-friendly UI

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure API keys and settings under **Taxi Booking > Settings**.
4. Use shortcode `[taxi_booking_form]` to place the booking form.
